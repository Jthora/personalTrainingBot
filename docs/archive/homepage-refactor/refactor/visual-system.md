# Visual System

## Header
- Simplify: logo + minimal nav; move profile/auth into Settings/Profile tab.
- Use coach accent sparingly (tab active state, chips) to reduce noise.

## Pages
- Plan: light surface with clear CTA block and concise cards; minimal borders.
- Cards: focused panel with subtle separators; timers prominent but not urgent.
- Progress: tile cards with consistent padding and hierarchy.
- Coach: accent-tinted panels showing palette preview and quotes.
- Settings: neutral surfaces for controls.

## Spacing and type
- Use consistent spacing scale; avoid cramming multiple widgets.
- Keep current type choices; ensure headings/buttons are legible on mobile.

## Signals
- Alignment chip only on Plan or Coach; do not repeat elsewhere.
- Badge visuals limited to Progress.
