# Responsiveness

## Tabs
- Horizontal scrollable tab row on mobile; consider dropdown fallback if many items.
- Active tab stays visible; avoid wrapping to multiple lines.

## Layouts
- Plan: two-column on desktop, single column on mobile; CTA block near top.
- Cards: single column; cards stack; timers and actions stay visible without horizontal scroll.
- Progress: tile grid collapses to single column under tablet width.
- Coach and Settings: single column lists on mobile.

## Scroll
- Prefer one main scroll per section; avoid nested scroll panes in sidebar/main split.
- Keep sticky header/tab bar minimal height to preserve vertical space.

## Touch targets
- Minimum 44px tap targets; adequate spacing for tab taps and CTA buttons.
