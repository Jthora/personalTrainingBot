# Wireframes (described)

## Tabs
- Top under header: horizontal tab bar (Plan, Cards, Progress, Coach, Settings). On mobile, scrollable or dropdown.

## Plan (desktop)
- Two-column layout: left main 70%, right rail 30%.
- Main: Today’s Plan banner -> Up Next card -> short upcoming list.
- Rail: optional alignment chip or quick tips.
- CTA block pinned near top: Start/Resume primary, Regenerate secondary.

### Plan (mobile)
- Single column stacking: banner, CTA block, Up Next, list.
- Tab bar scrolls; content is single-scroll.

## Cards (desktop)
- Single wide column. Card table with three slots stacked vertically.
- Each slot: header (label, timer), meta tags, content, toolbar (Hold, Share, Next).
- Optional slim filter/search row above table.

### Cards (mobile)
- Single column; slots stack; timers align top-right.

## Progress
- Grid of tiles: Streak/XP, daily goal, weekly goal, badges strip, achievements list.
- History graph placeholder if data exists.
- Single column stack on mobile.

## Coach
- Top: coach selector + palette preview.
- Middle: motivational quote/feed, coach tips.
- Bottom: alignment status and advisory.

## Settings/Profile
- List of toggles/cards: Web3 connect, profile info, cache/storage, sounds/notifications.
- Link to full settings page.
