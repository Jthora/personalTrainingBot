# Network State Telemetry

## Events
- offline_enter, offline_exit
- cache_hit, cache_miss (by resource type)
- sw_install, sw_activate, sw_update_available, sw_update_applied
- preload_triggered, preload_completed

## Metrics
- Offline success rate per route.
- Cache hit ratio for kits/drills.
- SW update adoption.

## Acceptance
- Event names documented; mapped to dashboards; included in test matrix.
