# Dashboards and Alerts

## Dashboards
- IA/Navigation: tab views, deep-link success.
- Readiness: render rate, time-to-ready, next-action CTR.
- Offline/SW: offline success, cache hit ratio, SW update adoption.
- Drills: completion vs abort, step drop-off.
- Signals/AAR: item lifecycle, AAR creation/export.

## Alerts
- Spikes in deep-link failures.
- Offline start failures above threshold.
- SW update failures; cache miss spikes.

## Acceptance
- Queries defined for each dashboard; alert thresholds documented.
