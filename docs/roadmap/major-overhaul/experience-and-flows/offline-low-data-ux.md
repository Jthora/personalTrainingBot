# Offline and Low-data UX

## Indicators
- Status badge for offline/online; cache state for kits/drills.

## Controls
- Preload/sync action; low-data toggle; clear cache option.

## States to Design
- Offline with cache; Offline missing data; SW updating; Cache eviction.

## Acceptance
- Copy and UI patterns for each state; telemetry for toggles and sync.
