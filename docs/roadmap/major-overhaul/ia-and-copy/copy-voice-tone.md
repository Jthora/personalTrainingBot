# Copy, Voice, and Tone

## Voice Anchors
- Calm, precise, operational; zero hype.
- Brief imperatives; avoid crypto/gym slang.

## Patterns
- Status: "Ready", "Sync required", "Offline, using cached intel".
- CTAs: "Start drill", "Review Signals", "Sync kits".
- Errors: actionable, low-drama: "Cache missing for this drill. Go online to sync.".

## SOP/ROE Messaging Guide
- Canonical guide: `docs/major-overhaul/ia-and-copy/sop-roe-tone-guide.md`
- Apply alert/prompt/guidance formats when editing system messages.

## Acceptance
- Copy kit covers headings, CTAs, empty/error/offline states for primary surfaces.
- Tone guide referenced in PRD/story review.
